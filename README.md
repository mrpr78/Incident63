## 🎮 How to Play: Quickstart Guide 🚦

1. **Install Roblox** and make sure you have an account.
2. **Search** for "Kill R63" in the Roblox games library.
3. **Join** a public server or **host** your own private match.
4. **Earn** starting credits by surviving the first 3 waves.
5. **Build** your base: walls 🧱, turrets 🔰, traps ⚡.
6. **Purchase** new gear: weapons 🔫, upgrades 🆙, and skins 👕.
7. **Communicate** with teammates via chat and quick pings.
8. **Survive** as many waves as possible, defeat epic bosses, and claim the high score!

> 💡 **Pro Tip**: Combine turret types for synergy: e.g., ice + cannon = crowd control + damage!
